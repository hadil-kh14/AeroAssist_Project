# AeroAssist 

**AeroAssist** is an intelligent aviation maintenance assistant designed to help technicians and engineers quickly diagnose mechanical failures (such as compressor leaks) directly from the field. 

The application provides a sleek web interface that turns natural language queries into actionable troubleshooting steps, reducing aircraft downtime.

---

##  Technologies Used

AeroAssist is built using a modern, lightweight AI stack:

*   **Frontend UI:** ![Streamlit] — Used to build the interactive web dashboard instantly with Python.
*   **AI Orchestration:** ![LangChain] — Used to manage prompts and handle seamless communication with Large Language Models (LLMs).
*   **LLM Inference API:** ![Groq] — Used for ultra-fast, real-time AI responses with zero latency.
*   **Core Model:** ![Meta Llama 3] — The state-of-the-art open-source model providing expert-level mechanical reasoning.
*   **Environment Management:** ![Python Dotenv] — Keeps private API keys secure and out of the public code repository.

---

##  Features
*  **Natural Language Search:** Ask complex maintenance questions just like talking to a senior human engineer.
*  **Blazing-Fast Answers:** Powered by Groq's LPU inference engine for near-instant responses.
*  **Field-Ready Design:** Responsive interface optimized for technicians using tablets or laptops in hangars.
